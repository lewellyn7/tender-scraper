Telegram 推送设置

配置说明

  • Bot Token: 从 @BotFather 获取
  • Chat ID: 使用 @userinfobot 获取
  • 最低预算: 只推送预算不低于此值的项目
  • 关键词过滤: 只有包含指定关键词才会推送
  • 推送阈值: 积累N条匹配项目后发送汇总通知

📚 RAGFlow 知识库配置

配置说明

  • RAGFlow 地址: RAGFlow 服务的 URL
  • API Key: 在 RAGFlow 界面 → 用户头像 → API Keys 获取
  • 知识库 ID: 创建知识库后的 ID（用于语义检索）
  • 修改后需重启服务生效

🤖 AI 模型配置

配置说明

  • 不启用：仅使用规则/正则提取，速度快但精度较低
  • OpenAI：精度高，需配置 API Key（自动路由到文档分析）
  • RAGFlow：使用本地 RAGFlow 服务，无需 API Key
  • 修改配置后需重启服务生效
