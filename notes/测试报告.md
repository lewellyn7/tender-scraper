## 三、代码修复摘要

### Bug 1: `_parse_date` 中文日期格式截断
**文件:** `app/crawlers/base.py`
**问题:** `date_str[:10]` 将"2024年05月10日"截断为"2024年05月10"导致匹配失败
**修复:** 检测格式字符串是否含中文年，使用完整字符串

```python
# 修复前
return datetime.strptime(date_str[:10], fmt)

# 修复后
date_part = date_str if '年' in fmt else date_str[:10]
return datetime.strptime(date_part, fmt)
```

### Bug 2: `_parse_datetime` 中文时间格式处理
**文件:** `app/crawlers/base.py`
**问题:** 替换逻辑错误（`时`→`:` 后"30分"残留）+ 格式匹配顺序错误（秒级格式在前优先匹配了不含秒的输入）
**修复:** 完善替换链 + 纠正格式列表顺序（短格式优先）

```python
# 修复后
s = re.sub(r'[年]', '-', datetime_str)
s = re.sub(r'[月]', '-', s)
s = re.sub(r'[日]', ' ', s)
s = re.sub(r'[时]', ':', s)
s = re.sub(r'[分]', '', s)
formats = [
    "%Y-%m-%d %H:%M",       # 短格式优先
    "%Y-%m-%d %H:%M:%S",
    "%Y/%m/%d %H:%M",
    "%Y/%m/%d %H:%M:%S",
]
for fmt in formats:
    try:
        return datetime.strptime(s, fmt)
    except Exception:
        pass
```
